# Celestial Navigation

An interactive 3D stellar navigation visualization built with Three.js, using real astronomical data.

## Overview

This project visualizes nearby stars in an explorable three-dimensional environment using data from the Gaia star catalog. Users can freely navigate through space, inspect stellar positions, and experiment with concepts related to interstellar navigation and astronomical mapping.

The long-term goal is to create an immersive navigation system that combines scientific accuracy with an intuitive visualization of our stellar neighborhood, and maybe some sci-fi spaceship HUD integration.

## Features

* Interactive 3D star field
* Real stellar positions from the Gaia catalog
* Thousands of rendered stars
* Free-flight camera controls
* Bloom post-processing effects
* Heads-up display (HUD)
* Built with Vite and Three.js
* Designed for future expansion with pulsars, nebulae, constellations, and coordinate grids

## Technologies

* Three.js
* JavaScript (ES Modules)
* Vite
* HTML5
* CSS3

## Installation

Clone the repository:

```bash
git clone https://github.com/GabrielleButler/<repository-name>.git
```

Navigate into the project:

```bash
cd <repository-name>
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
